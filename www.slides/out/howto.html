dj-deck is a system for creating beautiful presentation slides.

In this guide I'll show you how to write dj-deck slides.

Hit the right arrow or swipe left for the next slide.

Hit the 'n' key to see the notes.

I'll use the notes to explain how to write the slides. On the slides themselves you'll see the results illustrated.
Hit the right arrow or swipe left for the next slide.

Use '+slide' to create a new slide.

Use 'p' to create a paragraph.
